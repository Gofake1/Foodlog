#  Foodlog

Basically a UI wrapper for HealthKit nutrition information.

Features:
* Write to HealthKit
* Chronological, searchable history
* Barcode scanning (TODO)
* Sync over iCloud (TODO)

### Build
Tested on Xcode 9 and macOS 10.13.
